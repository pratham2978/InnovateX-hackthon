Problem Statement
What we set out to solve
SYNAPSE
💡 Engineering students today are overwhelmed by fragmented tools — they juggle 5–8 different platforms just to prepare for exams and placements, wasting time and losing momentum.

📚 Scattered Academic Resources

Students hop between YouTube, NPTEL, college PDFs, and Google Drive to access study material. There is no single organized system to break down a syllabus and identify high-priority topics.

💻 No Structured DSA Practice

Most platforms either paywall quality problems or lack visual learning. Students struggle to understand HOW algorithms work — not just the code. No platform combines visualizations, pattern recognition, and practice under one roof.

📄 Resume Blindness

Students submit resumes without knowing their ATS score or how well they match job descriptions. No free tool combines ML-based role prediction, skill gap identification, AND personalized study roadmaps.

🎤 Zero Interview Practice

Students only practice interviews the night before. Peer mock interview platforms are expensive, and AI interviewers rarely cover behavioral + DSA + system design in one session.

🧩 No Engagement Mechanism

Without motivation or progress tracking, students procrastinate. Standard platforms show stats but don't gamify the learning journey with XP, streaks, badges, and global leaderboards.

🏫 College Study vs Placement Gap

Students treat university exams and placement preparation as completely separate tracks, leading to burnout and poor time management. No platform bridges the academic-to-industry pipeline effectively.

Proposed Solution
One platform, complete student lifecycle
SYNAPSE

SYNAPSE — The Complete Student Intelligence Platform

A unified, AI-powered web ecosystem that covers a student's complete journey from university exams to top-tier tech company placements — all in one authenticated, gamified, and beautifully designed platform.

Instead of building yet another quiz app or LeetCode clone, SYNAPSE takes a full-stack, holistic approach — combining machine learning, generative AI, peer collaboration, and real-time analytics into a single coherent product with a premium experience students will actually want to use.

◆ Three Core Pillars
🏫

Academic Excellence

AI-powered syllabus analyzer, PYQ pattern detector, cheat sheet generator, and GPA tracker to ace university exams.

💼

Placement Readiness

ML resume analyzer, DSA visualizer, company-wise test banks, AI + peer mock interviews, and video rooms.

🎮

Motivation Engine

XP system, 7 unlock-able badges, daily login streaks, global XP leaderboard, and achievement notifications.

◆ How It Works — User Flow
  • 1
    Register / Demo Mode — Students sign up or try a full demo without any friction. JWT auth protects all routes.
  • 2
    Upload Resume — ML model classifies role, scores ATS compatibility, detects missing skills, and suggests personalized learning roadmaps.
  • 3
    Study for Exams — Upload syllabus PDF → Gemini AI generates topic tree, study roadmap, PYQ analysis, and an exam cheat sheet.
  • 4
    Practice DSA — Study patterns, watch animated visualizations, solve problems in-browser, and track progress with XP rewards.
  • 5
    Mock Interview — Match with peers for live video interviews, or practice with the AI Interviewer across 5 topics and 3 difficulty levels.
  • 6
    Climb the Leaderboard — Every action earns XP. Track streaks, earn badges, and compete on the global ranking board.
✅ SYNAPSE requires zero external subscriptions. It is a fully self-contained platform — a student can register, analyze their resume, study for exams, practice DSA, and prepare for interviews all in a single session without leaving the app.

◆ Design System Components
🌐 Neural Canvas

Animated particle network background on every page — interactive, responsive, and lightweight using HTML5 Canvas API.

💎 Glassmorphism Cards

backdrop-filter: blur(20px) + saturate(180%) with rgba backgrounds. Hover states glow with orange border shadows.

🌟 Ambient Glows

Blurred radial gradient blobs (filter: blur(130px)) create atmospheric depth behind content without affecting performance.

⚡ Spring Animations

cubic-bezier(0.34, 1.56, 0.64, 1) spring physics on all DSA visualizer bar transitions. Hover lift + scale effects on cards.

🔔 Toast Notifications

Slide-up toast system with color-coded borders for success (orange) and error (red). Auto-dismiss after 3.5 seconds.

📱 Fully Responsive

Desktop-first responsive design. Mobile hamburger menu, collapsible nav, grid layouts adapt to all screen sizes.

🅰️ Typography Scale

Plus Jakarta Sans (headings, 700-900 weight) + Inter (body, 300-600 weight) + Fira Code (monospace/code blocks).

🎯 Grid Overlay

Subtle 50px grid pattern fixed behind all content. Creates a technical, engineering aesthetic across the platform.

🌈 Code Highlighting

Active line highlighting in DSA visualizer — orange border-left + background fade when algorithm reaches that code line.
